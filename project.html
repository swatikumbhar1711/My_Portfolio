Online Art Gallary

INTRODUCTION
“Online Art Gallery” is an online website where people buy there painting from their favorite Artist. The website is contain all types of paintings. The basic need for this website is to increase the growth of the shop and spread awareness among the people about the shop so that the owner of the shop get more profit. There are many websites in which e commerce websites which allows a customer to buy the thing online. Here I am creating a website, which would particularly allowing the customer to buy the painting from the favorite artist. These all websites provide a payment option of cash on delivery. This project i.e. online art gallery is very helpful for customer where customer can buy directly paintings from home through internet connection where internet connection is must. This can be done through mobile phone or system.

Details in PROJECT


Front End



Front-End means a language that is used for user interface designing and coding. Front End should have following qualities:

• It must have a graphical user interface that assist employees that are not from I.T. background. • Easy to deploy and maintain. • Support to Object Oriented Programming.• Scalability and Extensibility.• Server and Oracle. • LightweightAccording to the above stated features I have selected PHP, JS as Front–End for developingmy project. JAVA is used in NeatBeans.






-------------------Back End-------------------





Back-End means a language that is used for database management. Back-End should have following qualities:

• Multiple user support. • Operating System compatible. • Stored procedures. • Easy to install. • Various drivers must be available. • Efficient data handling. • Provide inherent feature for security. • Easy to implement with Front-End. I will use SQL Server in Backend for developing my project. SQL server is more features and easily use for Database. It is also easy to implement SQL server with PHP.
